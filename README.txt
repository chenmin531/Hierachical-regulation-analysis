Hierachical regulation analysis-modification.py :
This script is designed to do Bayesian inference of kinetic models for identifying allosteric regulators with proteome, metabolome, and fluxome data, written with python 3.6, and depends on Hierachical regulation analysis-CCMS(openmebius).xlsx.


Hierachical regulation analysis-CCMS(openmebius).xlsx: 
Input file for the above script, which contains central carbon metabolism reaction models and corresponding enzyme abundance, reaction fluxes, and metabolome data.


Hierachical regulation analysis-modification.py is written with python 3.6 (in PyCharm Community Edition 2020.1.1) on Linux platform (Ubuntu 18.04.1), choose a location where you put the input file, and run the script. The running environment packages for the author is listed as follows:
